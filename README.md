# Dwarf Fortress Simulation - GoT
Un simulador político y económico inspirado en Dwarf Fortress y Game of Thrones, desarrollado en Python (Pygame).
El proyecto combina simulación social, gestión de tareas, sucesiones políticas y comportamiento emergente mediante estructuras de datos avanzadas como heaps, colas y diccionarios
